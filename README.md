# AI DevOps Agent Platform - Live Demo

🚀 **[View Live Demo](https://agenticdevops-demo.pages.dev)**

This is the interactive demo site for the AI DevOps Agent Platform - a multi-agent system that transforms natural language into production-ready cloud infrastructure.

## What You'll See

- **90-second terminal recording** showing the full workflow
- **Three demo scenarios:**
  1. Intent → Infrastructure (natural language to Terraform)
  2. Validation loop with intelligent error recovery
  3. Security policy enforcement (OPA gates)

## Features Demonstrated

- Intent-driven architecture with confidence tracking
- LangGraph multi-agent orchestration
- Terraform error intelligence with targeted replanning
- OPA security gates (blocks wildcards, open security groups, prompt injection)
- Human-in-the-loop approval for irreversible operations

## Tech Stack

- **Frontend:** Static HTML + Asciinema Player
- **Backend:** FastAPI + LangGraph + Redis
- **LLMs:** Claude Sonnet 4 (primary), GPT-4o (fallback)
- **Security:** Open Policy Agent (OPA)
- **Observability:** Prometheus + Jaeger + Grafana
